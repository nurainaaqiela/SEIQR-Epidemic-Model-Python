# SEIQR Epidemic Model Simulation

This project implements a compartmental SEIQR epidemiological model in Python.

## Model Features

- Susceptible (S)
- Exposed (E)
- Asymptomatic Infected (IA)
- Symptomatic Infected (IS)
- Quarantined (Q)
- Recovered (R)

## Analysis Included

- Basic Reproduction Number (R0)
- Threshold Behaviour (R0 < 1 vs R0 > 1)
- Impact of Quarantine
- Sensitivity Analysis

## Tools Used

- Python
- NumPy
- SciPy
- Matplotlib
